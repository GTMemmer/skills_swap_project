# campus_skillswap

A Django project for campus skills swapping and community collaboration.

## Setup

1. Activate the virtual environment:

```bash
cd '/Users/g.musicman/Documents/AI_ML/django project copilot'
source .venv/bin/activate
```

2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Apply migrations:

```bash
python manage.py migrate
```

4. Run the development server:

```bash
python manage.py runserver
```

## Project structure

- `manage.py` — Django management utility
- `campus_skillswap/` — main Django project configuration
- `MainApp/` — application module for core project features

## Notes

- This project uses Python 3.12 and Django 6.0.4.
- The development server is for local testing only.
